> 基于PBL（项目式学习）理念的职业角色扮演学习平台
> 适合小学1-6年级学生

## 🌟 核心功能

### 1. 注册定级
- 采集学生基本信息（昵称、性别、年级、学校、城市）
- 年级是核心字段，决定后续推送的内容

### 2. 角色选择
- 5个职业角色横向展示（上下错落）
- 每个角色配有卡通插画、图标、能力标签
- 悬停和点击效果

### 3. 🆕 项目三选一（v2.0新增）
- 展示该角色的3个不同维度项目
- 项目卡片包含：难度、挑战、使命、关键词
- 精美的交互动画和选中效果
- 学生自主选择感兴趣的项目

### 4. 项目详情确认
- 身份确认
- 项目背景任务说明
- 角色核心能力展示

### 5. 项目指南（PBL流程）
- **Step1 观察与发现**：引导观察、记录
- **Step2 学习与思考**：知识点+思考题
- **Step3 行动与创作**：动手实践
- **Step4 分享与迭代**：展示成果+获得勋章

## 🛠 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- **样式**：自定义CSS（Flexbox + Grid）+ 动画
- **PWA**：Manifest + Service Worker
- **部署**：GitHub Pages / 静态服务器
- **响应式**：Media Queries

## ⚠️ 当前限制

1. **年级数据**：目前只实现了三年级的完整数据（5角色×3项目）

3. **角色图片**：需要在`assets/images/roles/`目录下放置PNG图片
