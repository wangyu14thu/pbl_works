# PBL职业体验探索APP

基于PBL（项目式学习）理念的职业角色扮演学习平台，适合小学1-6年级学生使用。

## 项目简介

这是一个帮助小学生通过职业角色扮演完成跨学科探究项目的Web应用。学生可以选择不同的职业角色，接受相应的项目任务，并通过四个步骤（观察、思考、行动、分享）完成项目学习。

## 功能特点

### 核心功能

1. **用户注册与个人中心**
   - 基本信息录入（昵称、性别、年级、学校、城市）
   - 根据年级自动匹配适合的角色和项目

2. **角色选择**
   - 每个年级提供5个不同的职业角色
   - 角色包含：职业图标、名称、能力标签
   - 直观的卡片式选择界面

3. **项目匹配**
   - 基于（年级，角色）唯一匹配项目
   - 加载动画提供流畅的用户体验
   - 自动跳转到项目详情页

4. **项目指南（PBL四步法）**
   - **Step 1 观察与发现**：引导学生观察和记录
   - **Step 2 学习与思考**：提供知识胶囊和思考任务
   - **Step 3 行动与创作**：动手实践和创作
   - **Step 4 分享与迭代**：分享成果，获得勋章

### 交互功能

- 📸 图片上传（支持观察记录和作品展示）
- ✍️ 文本输入（支持文字描述和思考记录）
- 📝 多行输入列表（材料清单等）
- 🏅 勋章系统（完成项目获得专属勋章）
- 📊 进度可视化（四步进度导航）

## 使用方法

### 直接打开

1. 下载所有文件到同一目录
2. 双击打开 `index.html` 文件
3. 在浏览器中即可使用

### 本地服务器（推荐）

如果您想通过本地服务器运行：

```bash
# 使用Python 3
python3 -m http.server 8000

# 使用Python 2
python -m SimpleHTTPServer 8000

# 使用Node.js (需要安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 特色项目示例：自然守护者

### 项目背景
随着气温下降，校园里的流浪猫难以御寒。作为自然守护者，你的使命是利用环保材料，为它们策划并搭建一个温暖、隐蔽的家。

### 培养能力
- 🔍 深度观察：敏锐察觉环境变化
- ♻️ 绿色设计：环保材料创意改造
- ❤️ 共情行动：站在动物角度思考

### 学习步骤
1. 观察校园流浪猫的活动区域
2. 学习猫咪生活习性和材料知识
3. 设计并制作猫咪避风港
4. 分享成果，获得"初级自然守护者勋章"

## 功能扩展

### 添加新年级数据

编辑 `script.js` 文件，在 `rolesData` 和 `projectsData` 对象中添加新年级的数据：

```javascript
// 添加四年级数据示例
const rolesData = {
    3: [...], // 三年级数据
    4: [      // 四年级数据
        {
            id: 'R401',
            name: '新角色名称',
            icon: '🎯',
            projectId: 'P401',
            tags: ['能力1', '能力2', '能力3'],
            description: '角色描述'
        },
        // ... 更多角色
    ]
};
```

### 自定义样式

修改 `style.css` 中的CSS变量来调整主题颜色：

```css
:root {
    --primary-color: #6366f1;      /* 主色调 */
    --secondary-color: #8b5cf6;    /* 辅助色 */
    --success-color: #10b981;      /* 成功色 */
    /* ... 更多颜色变量 */
}
```
