# 角色选择页面设计说明 🎭

## 🎨 新设计特点

### 布局改进

#### ✅ 横向一排展示
- **5个角色卡片**横向排列在一行
- 每个卡片宽度：200px（固定宽度）
- 卡片间距：2rem（32px）
- 总体布局：使用 `flex` 布局，居中对齐

#### ✅ 上下错落分布
卡片通过 `margin-bottom` 创造高低起伏的视觉效果：

```
位置 1（自然守护者）：向上抬高 60px  🌿
位置 2（未来工程师）：基准线 0px    ⚙️
位置 3（少年CEO）：向上抬高 80px     💼  （最高）
位置 4（文化策展人）：向上抬高 20px  🎨
位置 5（游戏策划师）：向上抬高 40px  🎮
```

视觉上形成：**低 → 基准 → 高 → 中低 → 中高** 的波浪状排列

## 🖼️ 卡片结构

### 新的卡片布局（从上到下）：

```
┌─────────────────────┐
│   角色名称（顶部）    │  ← 渐变彩色文字
├─────────────────────┤
│                     │
│   卡通角色图片       │  ← 200x200px 区域
│                     │
├─────────────────────┤
│  标签1 标签2 标签3   │  ← 彩色标签
└─────────────────────┘
```

### 详细元素：

1. **角色名称**（顶部）
   - 位置：卡片最顶部
   - 字体大小：1.3rem
   - 样式：粉红到橙色渐变文字
   - 字重：900（超粗体）
   - 内边距：上下 1rem

2. **角色图片**
   - 位置：名称下方，居中
   - 尺寸：200x200px
   - 背景：淡彩色渐变（与角色主题匹配）
   - 图片来源：DiceBear API（卡通风格）
   - 备用方案：如果图片加载失败，显示 emoji 图标

3. **能力标签**
   - 位置：卡片底部
   - 样式：圆角徽章，带边框
   - 颜色：每个标签不同颜色
     - 标签1：蓝色系
     - 标签2：橙色系
     - 标签3：紫色系

## 🎭 卡通角色图片

### 图片生成方案

使用 **DiceBear Avatars API** 生成卡通风格头像：

```javascript
// API格式
https://api.dicebear.com/7.x/adventurer/svg
  ?seed={角色标识}
  &backgroundColor={背景色}
  &scale=90
```

### 每个角色的配置：

| 角色 | Seed | 背景色 | 主题色 |
|------|------|--------|--------|
| 🌿 自然守护者 | nature-guardian | #d1f4dd（淡绿） | 绿色 |
| ⚙️ 未来工程师 | engineer-future | #e3f2fd（淡蓝） | 蓝色 |
| 💼 少年CEO | young-ceo | #fff9e6（淡黄） | 黄色 |
| 🎨 文化策展人 | culture-curator | #ffe6f0（淡粉） | 粉色 |
| 🎮 游戏策划师 | game-designer | #f0e6ff（淡紫） | 紫色 |

### 图片特点：
- ✅ **卡通风格**：可爱的卡通人物
- ✅ **颜色匹配**：背景色与角色主题一致
- ✅ **自动生成**：无需手动设计
- ✅ **SVG格式**：矢量图，任意缩放不失真
- ✅ **备用方案**：图片加载失败时显示 emoji

## 🎨 视觉效果

### 彩色边框系统

每个角色卡片底部有**5px粗边框**，颜色与角色主题匹配：

```css
卡片1：#4cd964（绿色） - 自然守护者
卡片2：#5ac8fa（蓝色） - 未来工程师
卡片3：#ffcc00（黄色） - 少年CEO
卡片4：#ff6b9d（粉色） - 文化策展人
卡片5：#af52de（紫色） - 游戏策划师
```

### 光晕系统

每个卡片都有**对应颜色的发光阴影**：

- 基础阴影：深度感
- 彩色光晕：与底边颜色相同
- 悬停增强：光晕变亮变大

## ✨ 交互动画

### 悬停效果（Hover）：

1. **3D浮起**
   ```
   translateY(-25px)      → 向上浮起25px
   scale(1.1)             → 放大10%
   translateZ(40px)       → 3D景深效果
   ```

2. **图片放大旋转**
   - 图片放大1.1倍
   - 轻微旋转2度

3. **星星闪烁**
   - 卡片右上角出现 ✨
   - 快速闪烁动画

4. **光环扩散**
   - 从中心扩散的彩色光晕
   - 直径从200px扩大到350px

5. **标签弹出**
   - 所有标签轻微放大（1.05倍）
   - 增加阴影

6. **名称变色**
   - 从粉红-橙色渐变
   - 变为紫蓝色渐变

### 过渡动画：
- 时长：0.5秒
- 缓动函数：`cubic-bezier(0.175, 0.885, 0.32, 1.275)`
- 效果：弹性动画，略微超出目标然后回弹

## 📱 响应式设计

### 宽屏（> 1200px）
- 5个卡片横向一排
- 容器宽度：1400px
- 自由上下错落

### 中屏（768px - 1200px）
- 允许换行
- 居中对齐
- 保持错落效果

### 小屏（< 768px）
- 自动换行为多行
- 每个卡片统一间距
- 取消上下错落（都是2rem间距）

## 🎯 技术实现要点

### CSS关键技术：

1. **Flexbox布局**
   ```css
   display: flex;
   justify-content: center;
   align-items: flex-end;  /* 底部对齐，实现错落 */
   ```

2. **玻璃形态**
   ```css
   background: rgba(255, 255, 255, 0.95);
   backdrop-filter: blur(15px);
   ```

3. **3D变换**
   ```css
   transform-style: preserve-3d;
   transform: translateY(-25px) scale(1.1) translateZ(40px);
   ```

4. **渐变文字**
   ```css
   background: linear-gradient(135deg, #ff6b9d 0%, #ffa06b 100%);
   -webkit-background-clip: text;
   -webkit-text-fill-color: transparent;
   ```

### JavaScript关键逻辑：

```javascript
// 图片加载失败时的备用方案
onerror="this.style.display='none'; 
         this.parentElement.innerHTML='<div class=\"role-icon\">${role.icon}</div>';"
```

## 🚀 使用说明

1. **查看效果**：
   - 打开 `index.html`
   - 完成注册，选择"三年级"
   - 进入角色选择页面

2. **查看卡通图片**：
   - 图片会自动从 DiceBear API 加载
   - 每个角色有独特的卡通形象

3. **体验交互**：
   - 鼠标悬停在任意角色卡片上
   - 观察3D浮起、光晕、星星等效果

4. **点击选择**：
   - 点击任意角色卡片
   - 进入加载动画
   - 然后显示项目详情
