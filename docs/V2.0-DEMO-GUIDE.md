## 完整体验流程

### Step 1: 注册定级

**操作：**
1. 输入昵称（如：小明）
2. 选择性别
3. **重要：选择"三年级"**（目前只实现了三年级的数据）
4. 输入学校和城市
5. 点击"开始探索之旅"

**看到的效果：**
- 卡通风格的注册表单
- 明亮的未来宇宙空间背景

---

### Step 2: 选择角色

**操作：**
1. 欢迎页显示你的昵称
2. 看到5个角色横向排列（上下错落）：
   - 🌿 自然守护者
   - ⚙️ 未来工程师
   - 💼 少年CEO
   - 🎨 文化策展人
   - 🎮 游戏策划师
3. 点击任意角色卡片

**看到的效果：**
- 角色卡片悬停时向上浮起、缩放
- 角色图片会旋转和放大
- 点击后出现加载动画

---

### Step 3: 🆕 项目三选一（新功能！）

**操作：**
1. 加载动画后，进入**项目选择页面**
2. 看到顶部的角色徽章（带浮动动画）
3. 看到欢迎标题："你好，三年级的[角色名]！"
4. 看到引导语："地球总部发来 3 项紧急挑战，请选择一项你最想开启的使命："
5. 看到3个项目卡片横向排列

**以"自然守护者"为例，看到：**

#### 项目A：校园流浪猫避风港 ⭐⭐⭐
- **核心挑战：** 寒潮来袭，校园流浪猫缺少过冬居所。
- **你的使命：** 利用废旧泡沫箱和防水布，为猫咪设计并搭建一个温暖、隐蔽的"避风港"。
- **关键词：** 动物关怀、工程搭建、环保利用

#### 项目B：校园植物"身份证" ⭐⭐
- **核心挑战：** 同学们经常路过绿化带，却叫不出植物的名字。
- **你的使命：** 走访校园，查阅植物特性，为它们制作精美的、带有科普知识的互动"身份证"。
- **关键词：** 生物观察、科普写作、视觉设计

#### 项目C：雨水收集灌溉系统 ⭐⭐⭐⭐
- **核心挑战：** 校园花园在旱季需要大量自来水浇灌，造成水资源浪费。
- **你的使命：** 制作一个简单的雨水收集过滤器，尝试用自然雨水解决植物的"口渴"问题。
- **关键词：** 节约用水、物理实验、资源循环

**交互效果：**
- 悬停卡片：向上浮起、缩放、出现星星✨动画
- 点击卡片：绿色边框+绿色光晕（选中状态）
- 点击"查看详情并接受任务"按钮：进入项目详情页

**可选操作：**
- 点击"← 重新选择角色"返回角色选择页

---

### Step 4: 项目详情确认

**操作：**
1. 看到身份确认："[昵称]，[角色名]已就位"
2. 看到项目标题
3. 看到背景任务说明
4. 看到该角色的核心能力（3个）
5. 点击"确认上岗，获取指南"

---

### Step 5: 项目指南（PBL流程）

**操作：**
1. 看到4步进度条：观察 → 思考 → 行动 → 分享
2. 每一步包含：
   - 任务说明
   - 知识点（部分步骤）
   - 思考题输入框
   - 照片上传区
3. 完成任务，点击"下一步"
4. 最后一步完成后，获得勋章

## v2.0 核心变化对比

### 旧流程（v1.0）
```
注册 → 角色 → 加载 → [直接显示项目详情] → 指南
```
- 每个角色只有1个固定项目
- 学生没有选择权
- 共5个项目

### 新流程（v2.0）
```
注册 → 角色 → 加载 → [项目三选一页面] → 项目详情 → 指南
```
- 每个角色有3个不同维度的项目
- **学生可以自主选择**感兴趣的项目
- 共15个项目
- 项目难度可视化（星级）
- 项目信息更丰富（挑战、使命、关键词）

## 设计亮点展示

### 1. 项目卡片设计
- **磨砂玻璃质感**：白色半透明背景+模糊效果
- **渐变色头部**：紫色渐变（#667eea → #764ba2）
- **彩色关键词标签**：
  - 第1个：蓝色系
  - 第2个：橙色系
  - 第3个：绿色系
- **难度星级**：直观显示项目难度

### 2. 交互动画
- **卡片悬停**：
  - 向上浮起15px
  - 缩放至105%
  - 出现粉色光晕
  - 右上角出现旋转的⭐
- **选中状态**：
  - 绿色边框（#4cd964）
  - 绿色光晕
  - 按钮变为粉色渐变
- **角色徽章**：
  - 持续上下浮动（3秒循环）
  - 图标左右轻微旋转

### 3. 响应式适配
- **桌面端（>1024px）**：3列网格
- **平板端（768-1024px）**：2列或1列
- **移动端（<768px）**：单列，卡片全宽

## 注意事项

1. **年级选择**：目前只有三年级有完整数据，其他年级会显示空白
2. **角色图片**：需要确保`assets/images/roles/`目录下有对应的PNG图片
3. **浏览器兼容性**：建议使用Chrome、Edge、Safari最新版
4. **移动端测试**：可以在开发者工具中切换到移动设备视图

## 问题排查

### 问题1：项目选择页空白
**原因：** 角色的`projectIds`数组为空或项目数据不存在
**解决：** 检查`rolesData`和`projectsData`的对应关系

### 问题2：卡片样式错乱
**原因：** CSS文件未正确加载
**解决：** 确认`assets/css/style.css`路径正确

### 问题3：点击卡片无反应
**原因：** JavaScript事件未绑定
**解决：** 打开浏览器控制台查看错误信息
